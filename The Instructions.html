Colorful Buttons

Button numbering starts from top left, then top right, mid left, mid right, and so on. All the possible button color is either RED, GREEN, BLUE, YELLOW, MAGENTA, or CYAN

6 Buttons

IF the amount of red button(s) minus green button(s) is bigger than the amount of blue button(s) AND there is at least 1 green button(s), CLICK the first green button

OTHERWISE, IF there is more red button than cyan, CLICK the fourth button

OTHERWISE, IF there is a magenta button, CLICK the first magenta button

OTHERWISE, IF there is no cyan button, CLICK the blue button

OTHERWISE, CLICK the fifth button

5 Buttons

IF there is more than 1 blue button AND the second button is NOT red, CLICK the second button

OTHERWISE, IF there is NO red button, CLICK the cyan button

OTHERWISE, IF there is exactly 2 yellow buttons, CLICK the red button

OTHERWISE, IF the color of the first button is green OR the color of the fourth button is NOT green, CLICK the first button

OTHERWISE, CLICK the last button

4 Buttons

IF the amount of red button(s) is strictly bigger than green button AND there is exactly 1 yellow button, CLICK the yellow button

OTHERWISE, IF there is more than 1 yellow button, CLICK the last yellow button

OTHERWISE, IF the third button is cyan AND the second button is yellow, CLICK the third button

OTHERWISE, IF there is NO magenta button, CLICK the first button

OTHERWISE, IF there is exactly 1 red button AND more than 1 green button, CLICK the red button

OTHERWISE, click the second button
